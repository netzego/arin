An opinionated archlinux installer

### Usage
```sh
resolvectl query archlinux.org
timedatectl
mount -o remount,size=4G /run/archiso/cowspace
bash -x ./arin.sh $VOLUME |& tee -p $LOGFILE
```

### Files
- arin.authorized_keys
- arin.config
- arin.keyfile
- arin.packages
- arin.roothash
- arin.skeleton/
